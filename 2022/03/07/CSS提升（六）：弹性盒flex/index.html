flex(弹性盒、伸缩盒)

是css中又一种布局手段，它主要用来代替浮动过来完成页面的布局

flex可以使元素具有弹性，让元素可以跟随页面大小的改变而改变

弹性容器

要使用弹性盒，必须将一个元素设置为弹性容器

  • display：flex 设置为块级弹性容器（大多情况用这个）

    display：inline-flex 设置为行内弹性容器

  • flex-direction 指定容器中弹性元素的配列方式

    • 主轴：弹性元素的排列方向

      侧（辅）轴：与主轴垂直方向的轴

    • 可选值：

      • row 默认值 水平排列（左向右）

        主轴：自左向右

      • row-reverse 水平排列（右向左）

        主轴：自右向左

      • column 纵向排列（上向下）

      • column-reverse 纵向排列（下向上）

  • flex-wrap 设置弹性元素是否在弹性容器中自动换行

    • 可选值
      • nawrap 默认值，不会自动换行
      • wrap 元素沿着辅轴方向自动换行
      • wrap-reverse 元素沿着辅轴反方向换行

  • flex-flow wrap和direction的简写属性

    flex-flow：row wrap

  • justify-content 分配主轴上的空白空间

    • 可选值：
      • flex-start 元素沿着主轴起边排列
      • flex-end 元素沿着主轴终边排列
      • center 元素居中排列
      • space-around 空白分布到元素两侧
      • space-between空白分布到元素之间
      • space-evenly 空白分布到元素的单侧（兼容性不高）

  • algin-items 元素在辅轴上如何对齐

    • 可选值：
      • stretch 默认值 将元素的长度设置为相同的值（同一行的高度相同）
      • flex-start 元素不会拉伸，沿着辅轴起边对齐
      • flex-end 元素不会拉伸，沿着辅轴终边对齐
      • center 元素不会拉伸，居中对齐
      • baseline 基线对齐（文本基线）
    • 用于弹性元素：align-self 用来覆盖当前弹性元素上的align-items

  • align-content 分配辅轴上的空白空间

    • 可选值同justify-content

    弹性元素

弹性容器的子元素是弹性元素（弹性项），必须是直接子元素，后代不算

一个元素可以同时是弹性容器和弹性元素

  • flex-grow 指定弹性元素的伸展系数（当父元素有多余空间时，子元素如何伸展）

    • 父元素的剩余空间，会按照弹性元素的伸展比例进行分配
    • 默认值为0（不伸展）

  • flex-shrink 指定弹性元素的收缩系数（当父元素空间不足时，子元素如何收缩）

    • 默认值为1 (等比例收缩)

      值为0 （不收缩，溢出）

      缩减多少是根据 缩减系数 和 元素大小来计算

  • flex-basis 元素基础长度（相当于弹簧静止时的长度）

    • 如果主轴是横向，则该值指定元素的宽度

      如果主轴是纵向，则该值指定元素的高度

    • 可选值：

      • auto，参考元素自身的高度或宽度
      • 具体数值

  • flex 简写属性

    • 顺序：增长系数 缩减系数 基础长度；
    • 可选值：
      • initial：相当于flex：0 1 auto （默认值，不可伸展可收缩））
      • auto：相当于flex：1 1 auto （可伸展可收缩）
      • none：相当于flex：0 0 auto （弹性元素没有弹性）

  • order 弹性元素排列顺序

    • 数值越大排越后面
