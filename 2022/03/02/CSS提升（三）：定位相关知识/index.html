定位：一种更加高级的布局手段

通过定位可以将元素摆放到页面的任意位置

position属性

  • 可选值：
    • static 默认值，元素是静止的没有开启定位
    • ralative 相对定位
    • absolute 绝对定位
    • fixed 固定定位
    • sticky 粘滞定位

注：实际开发中，定位并不会大面积使用，整体上不用定位，定位通常用在细节上的微调

相对定位

position：relative

特点

  • 相对定位 若不设置偏移量，元素不发生任何变化

偏移量：开启定位后，偏移量可以用来设置元素位置（与margin不同：不会影响别的元素）

​ top、bottom：垂直方向定位（使用其一）

​ left、right：水平方向定位（使用其一）

*注：以上值要生效必须开启定位，若没开启定位，设置偏移量无效。

  • 相对定位 元素是参照于元素在文档流中的位置进行定位
  • 相对定位 会提高元素的层级
  • 相对定位 不会使元素脱离文档流
  • 相对定位 不会改变元素的性质（块还是块，行内还是行内）

形象理解：灵魂出窍，元素肉体（偏移前）还占据页面位置，元素灵魂（偏移后）挪动了

绝对定位

position：absolute

特点

  • 绝对定位 若不设置偏移量，元素的位置不发生任何变化

  • 绝对定位 元素是相对于其包含块进行定位

包含块（containing block）：

​ 正常情况下：包含块是离当前元素最近的祖先块元素

​ 绝对定位下：包含块是离当前元素最近的开启了定位的祖先元素；若所有祖先均未开启定位，则包含块是根元素。

  • 绝对定位 会提高元素的层级
  • 绝对定位 会使元素脱离文档流（不占据页面位置）
  • 绝对定位 会改变元素的性质（行内变成块，块的宽高被内容撑开）

绝对定位元素布局

水平方向布局

当开启绝对定位后，水平方向的布局等式需要添加left和right

left + margin-left + border-left + padding-left + width + padding-right + border-right + margin-right + right= 其包含块内容区的宽度

    可以调整auto的值：margin width left right

子元素在父元素中的定位：父元素相对，子元素绝对

  • 父元素相对：是为了在不脱离文档流的情况下开启定位（保证父元素周围的元素布局不混乱）
  • 子元素绝对：和其它子元素一起脱离文档流，避免互相影响

eg. 轮播图（父元素）中的原点（子元素）

固定定位

position：fixed

特点

  • 固定定位 是特殊的绝对定位，大部分特点和绝对定位相同
  • 唯一与绝对定位不同的地方：
    • 固定定位 元素是参考于浏览器的视口进行定位（不会随网页的滚动条移动，eg广告）

粘滞定位

position：sticky

特点

  • 粘滞定位 兼容性不太好（一般不采用，通常使用js来实现）
  • 粘滞定位 和相对定位特点基本一致
  • 与相对定位不同的地方：
    • 粘滞定位 可以在元素到达某个位置时将其固定（eg，导航栏）

元素的层级

对于开启了定位的元素，可以通过z-index属性来指定元素的层级

  • 语法：z-index：整数

    值越大元素的层级越高，越优先显示

  • 若元素层级一样，则优先显示html中结构靠下的元素

  • 祖先元素的层级再高也不会盖住后代元素
