基本语法

  • 声明变量：var a;

    声明+赋值：var a = 0;

  • 标识符：自主命名的（如变量名、函数名、属性名）

    • 命名规则：
      • 标识符中可含有字母、数字、_、$
      • 标识符不能以数字开头
      • 标识符不能是ES中的关键字或保留字
      • 标识符一般次啊用驼峰命名法
        • 首字母小写，每个单词开头字母大写，其余字母小写

  • 字符串：双引号或单引号都可以，但不要混用

  • typeof检查变量类型

    console.log(typeof a);

  • Number类型

    • Number.MAX_VALUE 最大值

      Number.MIN_VALUE 最小正值

      Infinity 正无穷

      -Infinity 负无穷

      NaN 特殊数字，表示Not A Number

  • Null类型

    • 表示空对象
    • typeof检查null时，返回object

  • Undefined类型

    • 当声明一个变量，但没有给它负值时，它的值为undefined

  • 强制类型转换

    • 将其它数据类型转换为String

      • toString() 注：null和undefined没有此方法
      • 调用String()函数 注：null和undefined可用此方法

    • 将其它数据类型转换为Number

      • 调用Number()函数

        • 字符串 –> 数字

          纯数字字符串 - 数字

          存在非数字 - NaN

          空串或全空格 - 0

        • 布尔 –> 数字

          true - 1

          flase -0

        • null –> 数字 0

        • undefined –> 数字 0

    • 将其它数据类型转换为Boolean

      • 调用Boolean()函数

        • 数字 –> 布尔

          除了0和NaN，其余都是true

        • 字符串 –> 布尔

          除了空串，其余都是true

        • 对象也会转换为true
