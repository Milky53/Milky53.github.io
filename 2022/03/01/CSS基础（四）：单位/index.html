长度单位

  1. 像素 （px）

    • 不同屏幕的像素大小不同，像素越小的屏幕显示效果越清晰
    • 同样的200px在不同的设备下显示效果不一样

  2. 百分比 （%）

    • 将其属性值设置为相对于其父元素属性的百分比
    • 设置百分比可以使子元素跟随父元素的改变而改变

  3. em

    • em是相对于元素（自身）的字体大小来计算

    • 1em = 1 font-size

      1 font-size = 16px（默认）

    • em会根据字体大小的改变而改变

    • 示例：

  4. rem

    • rem是相对于根元素（html）的字体大小来计算

    • 在移动端用得挺多

    • 示例：

  5. vw

    • 表示视口宽度（viewport width)

    • 100vw = 一个视口的宽度、1vw = 1%视口宽度

    • 与百分比不同：vw总是参考于视口宽度

      设计图的宽度：

      二倍图：750px，三倍图：1125px

    • 结合vm和rem来表示长度

颜色单位

  1. 颜色名

    • 比如：red、orange、yellow、blue、green……

  2. RGB

    • 语法：rgb(红，绿，蓝)

    • 取值范围：0~255

    • 特殊值：rgb（0，0，0）黑色

      ​ rgb（255，255，255）白色

  3. RGBA

    • 语法：rgb(红，绿，蓝，透明度)

    • 特殊值：rgb（176,199,167，1）完全不透明

      ​ rgb（176,199,167，0）完全透明

      ​ rgb（176,199,167，.5）半透明

  4. 十六进制的RGB

    • 语法：#红绿蓝

    • 取值范围：00~ff

    • 特殊值：#ff0000 红色

      ​ #00000 白色

    • 如果颜色两位两位重复可以简写：

      #aabbcc –> #abc

      #aabbcd 不能简写

  5. HSL、HSLA

    • H 色相：0~360
    • S 饱和度：0%~100% （0%灰）
    • L 亮度：0%~100%（0%黑，50%正，100%白）
    • A 透明度
