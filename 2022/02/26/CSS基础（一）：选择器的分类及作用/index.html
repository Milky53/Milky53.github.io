常用选择器

  • 元素选择器

    • 作用：根据标签名来选中指定元素

    • 语法：标签名{}

    • 示例：

  • id选择器

    • 作用：根据元素的id属性值选中一个元素（不能重复）

    • 语法：#id属性值{}

    • 示例：

  • 类选择器

    • 作用：根据元素的class属性值选中一组元素（可重复）

    • 语法：.class属性值{}

    • 示例：

  • 通配选择器

    • 作用：选中页面中的所有元素

    • 语法：*

    • 示例：

复合选择器

  • 交集选择器

    • 作用：选中同时符合多个条件的元素

    • 语法：选择器1选择器2……选择器n{}

    • 示例：

    • 注意：交集选择器中如果有元素选择器，必须使用元素选择器开头

  • 并集选择器（分组选择器）

    • 作用：同时选择多个选择器对应的元素

    • 语法：选择器1，选择器2，……，选择器n{}

    • 示例：

关系选择器

元素关系

  1. 父元素：直接包含子元素的元素
  2. 子元素：直接呗父元素包含的元素
  3. 祖先元素：直接或间接包含后代元素的元素（一个元素的父元素也是它的祖先元素）
  4. 后代元素：直接或间接被祖先元素包含的元素（子元素也是后代元素）
  5. 兄弟元素：拥有相同父元素的元素

示例：

  • 子元素选择器

    • 作用：选中指定父元素中的指定子元素

    • 语法：父元素 > 子元素{}

    • 示例：

  • 后代元素选择器

    • 作用：选中指定元素中的指定后代元素

    • 语法：祖先 后代{}

    • 示例：

  • 兄弟元素选择器

    • 选中下一个兄弟元素

      • 语法：前一个 + 后一个{}

      • 示例：

    • 选中下边所有的兄弟

      • 语法：兄 ~ 弟{}

      • 示例：

伪类选择器

伪类（不存在的类，特殊的类）

  • 伪类用来描述一个元素的特殊状态
    • 比如：第一个子元素、被点击的元素、鼠标移入的元素……
  • 伪类一般情况下都是使用：开头

示例：

  • :first-child

    • 作用：选中第一个子元素

    • 示例：

  • :last-child

    • 作用：选中最后一个子元素

    • 示例：

  • :nth-child()

    • 作用：选中第n个子元素

    • 特殊值：

      • n：全选中
      • 2n或even：选中偶数位的元素
      • 2n+1或odd：选中奇数位的元素

    • 示例：
