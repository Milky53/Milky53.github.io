关系型数据库与非关系型数据库

  • 关系型数据库（RDBMS）
    • Oracle
    • MySQL
    • ……
  • 非关系型数据库（NoSQL）
    • 键值型数据库：redis
    • 文档型数据库：MongoDB
    • 搜索引擎数据库：Solr、Elasticsearch、Splunk等
    • 列式数据库：HBase
    • 图形数据库

关系型数据库涉及原则

表、记录、字段

  • ORM思想（Object Relational Mapping）
    • 数据库中的一个表 <——-> Java或Python中的一个对象
    • 表中的一条数据(记录) <——-> 类中的一个对象（或实体）
    • 表中的一个列 <——-> 类中的一个字段、属性（field)

表的关联关系
