SQL的分类

  • DDL：数据定义语言。（对结构）
    • CREATE、ALTER、DROP、RENAME、TRUNCATE
    • 补充：DDL的操作一旦执行，就不可回滚。
  • DML：数据操作语言。（对数据）
    • 增删改查：INSERT、DELETE、UPDATE、SELECT
    • 补充：DML的操作默认清空，一旦执行，也是不可回滚的；但是如果在执行DML之前，执行了SET autocommit = FALSE,则执行的DML操作就可以实现回滚
  • DCL：数据控制语言。
    • COMMIT、ROLLBACK、SAVEPOINT、GRANT、REVOKE

基本的SELECT语句

  • 格式：SELECT 字段1,字段2,... From 表名
  • *：表中所有的字段（列）

列的别名

  • AS：alias别名，可以省略
  • 格式：SELECT 字段1 别名1,字段2 AS 别名2,字段3 "别名3"... From 表名
